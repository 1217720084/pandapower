#############################
Generic Networks
#############################
.. _ppNetworks:


Besides creating your own grids through the pandapower API pandapower provides generic networks through the networks module.
The pandapower networks modul contains simple test networks, randomly generated networks, CIGRE test networks, IEEE case files and generic networks from the dissertation of Georg Kerber.
